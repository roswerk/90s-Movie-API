Business Logic URL HTTP Method Request Body data Format Response Body data Format
Retrieve a list of ALL movies /movies GET - { "title": "Eyes Wide Shut",
"year": "1999",

Retrieve data about a single movie by TITLE /movies/:title GET - { "title": "Lolita",
"year": "1962",
"gender": "Drama" }
Return data about a genre (description) by name (e.g., “Drama”) /genre/:name GET - { "_id": 2,
"name": "Drama",
"description": "The drama genre ..." }
Return data about a director (bio, birth year, death year) by name /directors/:name GET - { { "_id": 4,
"name": "James Cameron",
